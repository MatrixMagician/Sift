    Create `pipeline/salience.py` with the same pure-module header as `cluster.py` (typer-free, print-free, SQL-free; the caller passes in already-queried clusters and groups). Copy the `_SEVERITY_RANK` dict verbatim from `cluster.py:56-66` (frozen vocabulary — never lexicographic). Add module-constant weights `_W_SEVERITY = 0.35`, `_W_COUNT = 0.20`, `_W_BURST = 0.15`, `_W_NOVELTY = 0.10`, `_W_PROXIMITY = 0.20` (sum 1.0; hand-tuned start per SPEC OQ4) and a `_SPAN_FLOOR` seconds constant plus a `_TAU_FALLBACK` decay seconds constant. Implement `rank_clusters(clusters: list[Cluster], groups: list[TemplateGroup], *, incident_time: datetime | None, since: datetime | None = None, until: datetime | None = None) -> list[tuple[Cluster, float]]`. Build a `{template_id: TemplateGroup}` index; for each cluster gather its member groups via `cluster.template_ids`. Parse member `first_ts`/`last_ts` with `datetime.fromisoformat` (skip None). Derive `incident_time` when None from the max member `last_ts`. Compute per-cluster raw features: `severity = _SEVERITY_RANK.get(cluster.severity_max, 0) / 5`; `count` log-damped as `log1p(cluster.count) / log1p(max_count)` across the candidate set; `burstiness = cluster.count / max(span_seconds, _SPAN_FLOOR)` where span = cluster_last − cluster_first, then min-max normalised across candidates; `novelty = exp(-abs(cluster_first − incident_time)/tau)`; `proximity = exp(-abs(cluster_last − incident_time)/tau)` with `tau` = case span / 4 or `_TAU_FALLBACK` if degenerate. Missing-timestamp clusters get burstiness=novelty=proximity=0. Score = the weighted sum. Apply the since/until window filter BEFORE scoring (drop clusters whose member span does not intersect the window; naive datetimes are treated as UTC by the caller). Sort by `(-score, cluster_id)`. Keep everything type-hinted (modern `list[...] | None`) and add British-English docstrings.
