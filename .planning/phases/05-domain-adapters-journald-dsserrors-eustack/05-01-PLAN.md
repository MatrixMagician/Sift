<objective>
Pay down the Phase-1 `isinstance(GenericLogAdapter)` coupling in `cli.py` by introducing a shared `base.ConfigurableAdapter` base class and promoting `to_utc` / `tz_override_for` into `base.py`. Output: `base.ConfigurableAdapter`, promoted `base.to_utc`/`base.tz_override_for`, generalised cli.py, a non-vacuous-coverage regression test, and ADR 0006.
</objective>

**As an** engineer triaging a MicroStrategy or systemd incident on an air-gapped host, **I want to** run `sift ingest` over journald JSON exports, DSSErrors logs (and their rotated `.bak` siblings), and EU-stack thread dumps, **so that** real production diagnostics become canonical UTC-ordered events â€” with nothing dropped silently and multi-node causality never inverted.
