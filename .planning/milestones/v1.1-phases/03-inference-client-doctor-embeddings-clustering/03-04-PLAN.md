<objective>
Implement `sift doctor` as a fail-fast dependency-ordered health check that verifies both inference
endpoints with real round-trips (including an actual embedding call), reports model IDs, checks the
returned dimension against any existing case index, verifies sqlite-vec loads, and warns on
determinism-breaking server config — stopping at the first critical failure with a non-zero exit.

Purpose: LLM-03 is the phase's headline UAT criterion. The real-embedding round-trip is what catches
Lemonade's OGA/ONNX recipe (chat works, embeddings silently don't) — a failure mode that
`/v1/models` cannot reveal. The --i-know-what-im-doing flag wires the LLM-02 SSRF override.

Output: `src/sift/cli.py` doctor command (replacing the stub) + the --i-know-what-im-doing / --model
flags; `tests/test_doctor.py` driving the whole sequence against a fake server (plus a live opt-in).
</objective>
