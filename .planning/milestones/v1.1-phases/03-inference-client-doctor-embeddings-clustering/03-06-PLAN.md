<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| clusters.label (model text) -> terminal | Model-generated labels are untrusted and rendered by show clusters |
| CLI flag -> client allow_public | --i-know-what-im-doing is the SSRF break-glass for analyze |
| interrupted run -> case.db | A crash/exception mid-embed must not leave partial state |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-20 | Spoofing | adversarial cluster label with control/bidi bytes rendered by show clusters | medium | mitigate | whole-line _sanitise on the rendered clusters line (WR-01 precedent); label length-capped upstream |
| T-03-21 | Information Disclosure | analyze pointed at a public endpoint | high | mitigate | client construction runs _assert_local; refusal unless --i-know-what-im-doing; tested |
| T-03-22 | Integrity | interrupted embed leaves partial vectors/clusters | medium | mitigate | all persistence inside one store.transaction(); test asserts unchanged state after a mid-embed raise |
| T-03-23 | Denial of Service | untrusted filename/label in a rich renderable | low | mitigate | static progress description only; DB/server text flows through _sanitise'd plain prints (T-02-06) |
</threat_model>
