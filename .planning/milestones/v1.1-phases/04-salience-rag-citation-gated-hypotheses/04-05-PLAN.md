## Phase Goal

**As an** on-call engineer triaging a production incident, **I want to** run `sift analyze` and get ranked, evidence-cited root-cause hypotheses, **so that** I can trust every claim because it cites a real stored event the model was actually shown — never a fabricated one.

*(Derived faithfully from the ROADMAP Phase 4 goal line; not invented scope.)*

<objective>
Complete the vertical slice at the CLI: extend `sift analyze` with `--hint/--since/--until/--top-clusters`, chain salience + hypothesise after the existing clustering leg, and map the run Outcome to the 0/3/1 exit-code contract. Un-stub `sift show hypotheses` to render persisted hypotheses (sanitised). Record the exit-code ADR.

Purpose: this is where the core value becomes user-observable — `sift analyze <case>` produces ranked, cited hypotheses and a scriptable exit code; `sift show hypotheses` inspects them.
Output: extended `analyze`, un-stubbed `show hypotheses`, the exit-code contract documented in `--help` and an ADR, plus CLI + e2e tests.
</objective>
